# Keywords

``` lua
and or not
nil true false
if then else elseif
for while do repeat until break
in local
return end function
```

- variable = nil
- 0 = true
- "" = true
- indexes starts from 1

# loops

``` lua
while(condition)
do
    statement(s)
end

for init, target_value, step
do
    statement(s)
end

repeat
    statement(s)
until(not condition)
```

# decision making

``` lua
if(condition)

then
    statement(s)
elseif (condition)
then
    statement(s)
else
    statement(s)
end
```
